◆ focusLost

bool UnityXRDisplayState::focusLost

Set this value to true when the display is not active or is not rendering the application content. Return false when the display becomes active and it is rendering content.

Optional legacy native pointer which will be exposed via XRDevice.GetNativePtr() c# api. Note that this is only to be used in legacy cases, it's better to pinvoke into your dll from c# to do implementation specific things.
