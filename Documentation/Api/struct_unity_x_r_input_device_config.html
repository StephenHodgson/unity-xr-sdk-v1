UnityXRInputDeviceCharacteristics characteristics
 The characteristics of the device, identifying how the device is used and what it can report.
 

Detailed Description

This represents what you can change on the device definition without disconnecting the device. If the provider calls UnityXRInputInterface::InputSubsystem_DeviceConfigChanged, Unity will call back with IUnityXRInputProvider::UpdateDeviceConfig and supply a config for you to modify. Developers will be notified through the InputDevices.DeviceConfigChanged event.
