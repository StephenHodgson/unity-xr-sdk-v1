Detailed Description

Implement this interface for your mesh provider. A mesh provider will typically provide multiple meshes, addressable by UnityXRMeshId.

Field Documentation

◆ userData

void* UnityXRMeshProvider::userData

Plugin-specific data pointer. Every function in this structure will be passed this data by the Unity runtime. The plugin should allocate an appropriate data container for any information that is needed when a callback function is called.

UnitySubsystemErrorCode( * UnityXRMeshProvider::AcquireMesh) (UnitySubsystemHandle handle, void *userData, const UnityXRMeshId *meshId, UnityXRMeshDataAllocator *allocator)

Invoked from any thread by Unity to acquire a particular mesh by id. This method should fill out a UnityXRMeshDescriptor using the provided allocator. If the plugin owns the mesh memory, it should not mutate the data until ReleaseMesh is called (see below).

◆ SetMeshDensity

UnitySubsystemErrorCode( * UnityXRMeshProvider::SetMeshDensity) (UnitySubsystemHandle handle, void *userData, float density)

Invoked by Unity on the main thread to request a change in the density of the generated meshes. Density level will be given as a value in the range [0.0-1.0] and the provider will determine how best to map that value to their implementation.

Setting this value does not guarantee an immediate change in the density of any currently created mesh and may only change the density for new or updated meshes.

Parameters
handleHandle obtained from UnityLifecycleProvider callbacks.
userDataValue of userData field when provider was registered.
densityThe requested density of generated meshes. Defined as a float in the range [0.0-1.0]
Returns
A UnitySubsystemErrorCode indicating success or failure. kUnitySubsystemErrorCodeSuccess Returned if everything is ok. kUnitySubsystemErrorCodeInvalidArguments Returned if the density level falls outside the expected range. kUnitySubsystemErrorCodeNotSupported Returned if the provider does not support this API in their implementation. kUnitySubsystemErrorCodeFailure Returned for ny other error or issue not defined above.
