Detailed Description

Unity sends this to the provider when it needs to know the haptic capabilities of a specific device. UnityXRInputProvider::HandleEvent's buffer parameter points to an object of this type when the event type is kUnityXRInputEventTypeHapticGetCapabilities. The provider fills in this structure for Unity to process.
