UnitySubsystemErrorCode( * IUnityXRDisplayShadingRateExt::CreateTextureWithShadingRate) (UnitySubsystemHandle handle, const UnityXRRenderTextureDesc *colorDepthDesc, const UnityXRTextureData *shadingRateTexture, UnityXRRenderTextureId *outTexId)

Provides the same functionality as IUnityXRDisplayInterface::CreateTexture, but also allows for supplying a shading rate image. This only works with the Vulkan graphics API if VK_EXT_fragment_density_map is available.

Parameters
[in]handleHandle obtained from UnityLifecycleProvider callbacks.
[in]colorDepthDescDescriptor of the color/depth textures to be created.
[in]shadingRateTextureNative texture pointer for a shading rate image.
[out]outTexIdTexture ID representing a unique instance of a texture.
Returns
kUnitySubsystemErrorCodeSuccess Successfully requested creation of texture.
kUnitySubsystemErrorCodeInvalidArguments Invalid / null parameters
kUnitySubsystemErrorCodeFailure Error
