◆ UnityXRMeshTopology

enum UnityXRMeshTopology

The type of mesh data, e.g., points, triangles, quads, etc. This affects how the vertex and index data are interpreted.

Enumerator
kUnityXRMeshTopologyTriangles 

The mesh data represents triangles, with 3 indices per triangle.

kUnityXRMeshTopologyTriangleStrip 

The mesh data represents a triangle strip. There are 3 indices for the first triangle, then every point after that is another triangle.

kUnityXRMeshTopologyQuads 

The mesh data represents quads, with 4 indices per quad.

kUnityXRMeshTopologyLines 

The mesh data represents lines, with 2 indices per line.

kUnityXRMeshTopologyLineStrip 

The mesh data represents a line strip. The are 2 indices for the first line, then every index after that is another line.

kUnityXRMeshTopologyPoints 

The mesh data represents individual points, with 1 index per point.
