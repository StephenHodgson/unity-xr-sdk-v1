◆ deviceAnchorToEyePose

UnityXRPose UnityXRNextFrameDesc::UnityXRRenderPass::UnityXRRenderParams::deviceAnchorToEyePose

Eye pose in device anchor space. This is usually a static transform that may change for IPD or foveated rendering. Head tracking is applied through the input subsystem by filling out kUnityXRInputFeatureUsageCenterEyePosition. Pose is in Unity coordinates (forward z is positive).
