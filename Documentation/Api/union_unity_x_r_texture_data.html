Detailed Description

Container for different ways of representing texture data. If the format (UnityXRRenderTextureDesc::colorFormat or UnityXRRenderTextureDesc::depthFormat) is a 'Reference' format, referenceTextureId must be set. Otherwise, nativePtr will be used.

void* UnityXRTextureData::nativePtr

Native texture ID if you've allocated it yourself. The texture ID varies by graphics API.

  • GL: texture name that comes from glGenTextures
  • DX11: ID3D11Texture2D*
  • etc.

You can pass in kUnityXRRenderTextureIdDontCare and have Unity allocate one for you.
