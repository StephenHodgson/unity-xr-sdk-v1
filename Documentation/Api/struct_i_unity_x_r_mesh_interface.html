◆ MeshInfoAllocator_Allocate

UnityXRMeshInfo*( * IUnityXRMeshInterface::MeshInfoAllocator_Allocate) (UnityXRMeshInfoAllocator *allocator, size_t count)

Allocates count UnityXRMeshInfo. Unity will periodically invoke your plugin's GetMeshInfos function. Your plugin should then call this function to allocate an array of UnityXRMeshInfo and populate them.

Parameters
countThe number of meshes currently tracked.
Returns
An array of UnityXRMeshInfo which the caller should populate with mesh info.

Allocates mesh data according to the attributes specified in attributes. Use this method if you want Unity to manage the mesh memory.

Parameters

triangleCountThe number of triangles to allocate.
indexFormatThe UnityXRIndexFormat indicating 16 or 32 bit indices

◆ MeshDataAllocator_SetMesh

void( * IUnityXRMeshInterface::MeshDataAllocator_SetMesh) (UnityXRMeshDataAllocator *allocator, const UnityXRMeshDescriptor *meshDescriptor)

Sets the mesh descriptor to use. Use this method if the plugin manages the memory. The buffers pointed to by mesh must exist until Unity invokes ReleaseMesh.

Parameters
meshDescriptorThe mesh descriptor for a mesh. Unused vertex attributes should be null. positions must be non null.

◆ MeshDataAllocator_SetUserData

void( * IUnityXRMeshInterface::MeshDataAllocator_SetUserData) (UnityXRMeshDataAllocator *allocator, void *userData)

(Optional) Set user data associated with this mesh. The userData pointer will be passed back to the provider's ReleaseMesh method. Use this to store additional data required to release the mesh.

Parameters
userDataA pointer which will be passed to ReleaseMesh
